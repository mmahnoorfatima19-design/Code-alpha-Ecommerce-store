# Code-alpha-Ecommerce-store
"A full-stack E-commerce Store built for the CodeAlpha Full Stack Development Internship (Task 1). Features include user login/register, dynamic product catalog with search, interactive shopping cart, order processing, and a protected Admin Dashboard. Developed using HTML, CSS, JavaScript, Node.js, Express.js, and MySQL."
## ✨ Features

* **User Authentication:** Secure registration and login system.
* **Product Catalog:** Dynamic product listings with category filtering and live search.
* **Shopping Cart & Checkout:** Add items, manage cart quantities, calculate total prices, and process orders.
* **Admin Dashboard:** Protected admin panel to monitor stats (visitors, total orders, completed orders, messages), update order statuses, and manage data.
* **Interactive UI:** Responsive modern design featuring a Black Friday typewriter effect and a visitor counter middleware.

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript (Vanilla JS)
* **Backend:** Node.js, Express.js
* **Database:** MySQL (`mysql2`)
* **Storage:** Browser `localStorage` for session authentication and cart persistence
